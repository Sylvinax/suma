View log files

The connector generates log files while synchronizing items. To get an overview of the artifacts synchronized, look at ccf-info.log. For detailed troubleshooting output, see ccf-debug.log.

Note: Log filles can be very large -- up to 10 MB.
  • Double-click a log file or right-click it and select Open in text editor. The file will open in the central editing area -- it may take a while for the file to download. You can search within the file and cut and paste the sections you're interested in.
  • Right-click a log file and select Open in console. The log file will open in the console view. Since this view updates periodically, it is a convenient way to monitor current synchronization activity.
