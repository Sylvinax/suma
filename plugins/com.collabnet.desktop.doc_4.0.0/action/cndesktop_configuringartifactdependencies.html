Make a tracker artifact depend on another artifact

To organize your work with tracker artifacts, you can make an artifact a "child" or a "parent" of another artifact.

  • A child artifact can have only one parent.
  • A parent artifact can have any number of children.
  • A parent artifact cannot be closed if a child is open.
  1. In the CollabNet Sites view, expand an artifact and select either of the these nodes:
    • Right-click Parent Dependency to specify a parent artifact for this artifact, and choose Create dependency.
    • Select Children Dependencies to create child dependencies for this artifact, and choose Create dependency.
  2. In the Create Dependency window, enter the ID or browse to the artifact(s) you're relating.
  3. Enter a comment that describes the relationship and click OK.
Tip: You can also expand a query in the CollabNet Sites view and drag and drop artifacts to create dependencies.
To delete a dependency, right-click a parent or child artifact and select Remove dependency.
