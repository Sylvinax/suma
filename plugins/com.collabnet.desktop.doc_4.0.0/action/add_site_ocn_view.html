Add a site to the openCollabNet view

To search for content on a site that runs CollabNet Community Edition, add the site to the openCollabNet view.

  1. In the openCollabNet view, click Configure Sites. The Configure Search Sites dialog appears.
  2. If you want to include a pre-defined site, for example, tigris.org, select it in the Configure Search Sites dialog and click OK.
  3. If you want to add a new site, click Add Site and do the following:
    1. Enter a name for the site.
    2. Enter the URL to access the site.
    3. Set the scope for the search. You can extend it to all projects on the site or limit it to specific projects.
    4. Click OK.
    5. In the Configure Search Sites dialog, select the new site and click OK.
The site should be available in the Search site dropdown in the openCollabNet view.
