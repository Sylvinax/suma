Monitor an item

You can see the list of users monitoring a TeamForge item. To get notified by email whenever the item changes, add yourself to the list.

These are some examples of items you can monitor:
  • Documents
  • Tasks
  • Tracker artifacts
  • Discussion forum topics
  • Files in a release
  1. In the CollabNet Sites view, right-click an item and select Show Users Monitoring. Users monitoring the item are listed.
  2. To begin monitoring the item, click Add.
  3. Add your name to the Users to Add list and click OK.
