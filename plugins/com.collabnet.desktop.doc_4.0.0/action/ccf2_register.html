Register the connector with Quality Center 11

This step is required if you are using Quality Center 11. To be able to connect to Quality Center, you need to install the Quality Center drivers on the system where the TeamForge connector server (CCF master) is installed.

On the connector server machine, visit http://<qchost>/qcbin/start_a.jsp?common=true.
