Add an external RDP client

You can specify and store a command to connect to a Windows host using the Remote Desktop Protocol.

  1. In the CollabNet Sites view where you have added a Lab Management site, right-click, and select Open external RDP client.
  2. In the Add CollabNet Site dialog, select Add Site.
  3. In the Open External RDP Client window, enter the command to connect to the client.
    For example, if you are on a Windows system and want to connect to a Windows XP host, enter a command in this format: 
    mstsc.exe /v:$HOST:$PORT
    .
  4. Select Prompt if you want this command to be displayed every time you try to open the client.
  5. Click OK.
    Note: The command gets stored under Windows > preferences > CollabNet Lab Management.
