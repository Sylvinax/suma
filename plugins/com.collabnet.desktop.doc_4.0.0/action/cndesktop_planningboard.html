Manage your sprint with the planning board

The planning board is the hub for your TeamForge project's Agile planning activities. It enables you to plan and monitor the features that are required in each sprint (or iteration) in order to release your product. In the planning board, tracker artifacts are represented as cards and displayed in ranked sequence. You can drag and drop cards to assign them to planning folders or change their rank.

Once you've planned the stories that will be tackled in a sprint, team members can subsequently use the task board to break down the stories into tasks, and track progress as they work on those tasks and complete them.

Introduced in version 3.5 of the CollabNet Desktop, the planning board replaces the project plan editor available in earlier releases. It provides all the functionality that came with that tool plus much more.

  • Configure the planning board
    As a TeamForge project administrator, you can configure multiple planning boards for a project. You might do this if you want to restrict the tracker types that can be displayed, or if you want to configure the layout of the fields that can appear on the artifact cards.
  • Use the planning board
    In the planning board, you can assign artifacts to planning folders or rank them using drag and drop, make quick changes and in-place updates, and view charts that help you monitor progress.
