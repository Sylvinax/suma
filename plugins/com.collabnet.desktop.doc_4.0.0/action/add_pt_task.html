Create an artifact

To create a Project Tracker artifact, select the artifact type, and enter the artifact attributes.

A Project Tracker artifact type is a template that provides a set of unique attributes and options to describe an artifact. Individual entries of a particular type are referred to as artifacts.
  1. In the CollabNet Sites view, right-click a CollabNet Enterprise Edition project that uses Project Tracker and select New > Task .
  2. In the New Project Tracker Report window, select an artifact type.
    Note: To make sure that the list displays all artifact types available in the repository, click Update artifacts from repository.
  3. Click Finish. The editor displays the fields for the new artifact.
  4. In the New Task screen, fill in the basic fields for the artifact.
    1. Enter a brief, distinguishing description for the artifact.
    2. Expand the Attributes section. Fill in the text fields and select appropriate values from the dropdown boxes.
    3. Click Submit.
