Lock a document

While you are working on a document, you can lock it to prevent others from editing it and then posting new versions to your TeamForge site.

When a document is locked, any user with the document view permission can view or download the document, but only document administrators and the document creator can post new versions to TeamForge.

You can lock a document when you create it or at any time afterward.

Note: Locking a document does not prevent it from being edited in its native application.
  1. In the CollabNet Sites view, right-click a TeamForge document and select Update Document.
  2. In the Update Document window, confirm that the lock status is currently Unlocked.
  3. Select Lock Document.
  4. Click OK.
The document is now locked.
