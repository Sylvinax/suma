Use Mylyn for your tasks

With CollabNet Desktop - Eclipse Edition, it's easy to work on CollabNet project tasks in Mylyn views.

To open a Mylyn view, use the Eclipse Window > Show View > Other > Mylyn option.
  • Add a task repository
    To track TeamForge tracker and other CollabNet project tasks using Mylyn's task lists, start by adding the repository to the Task Repositories view.
  • Create a TeamForge task list
    After you add your TeamForge repository to the Mylyn Task Repositories view, you can create task lists for your tasks, tracker artifacts or planning folder artifacts by adding queries.
