Add a CloudForge site

Whether you're a developer or project manager, you can enhance your CloudForge experience by adding your site to your CollabNet Desktop — for the Eclipse or Microsoft Visual Studio IDE, or for Microsoft Windows.

  1. In the CollabNet Sites view, click the Add CollabNet Site icon.

  2. In the Add a CollabNet Site dialog, select CloudForge for the site type and click Next.
  3. Enter a description for the site and click Next. The description is displayed as the site name in the CollabNet Sites view.
  4. Provide your username, password and domain (for example, the name of your organization) for the CloudForge site. If you don't have a CloudForge account, you can sign up here.
  5. Click Finish.
Your CloudForge site appears in the CollabNet Sites view.
