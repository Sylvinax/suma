Update a task

You can edit an artifact or issue from your task list or from the task repository.

At any time after an issue or artifact is created, you can edit it if you have the appropriate permissions. For example, if an issue is assigned to you, you can edit it to update the status from Open to Fixed, or you can add a comment regarding work completed or additional information required.
  1. Open a task in one of these ways
    • If the task is in your Task List, double-click it.
    • If the task is not in your Task List, right-click a task repository, select Open Repository Task, enter the ID and click OK.
    The artifact or issue appears in the task editor.
  2. Make the appropriate changes: update an attribute field, change a user related to the task, for example, the user "Assigned to" the task, add an attachment, or enter a comment.
  3. When you have finished editing your task, click Submit for your changes to take effect.
