Work with Project Tracker

In the CollabNet Sites view, you can create new artifacts and run queries.

  • What is Project Tracker?
    Project Tracker is a highly configurable artifact tracking tool that meets the needs of different types of projects as well as individual users.
  • Find Project Tracker artifacts
    To search for Project Tracker artifacts, run a pre-defined query.
  • Create an artifact
    To create a Project Tracker artifact, select the artifact type, and enter the artifact attributes.
  • Add an attachment
    You can attach text information from the clipboard, or files within or outside your Eclipse workspace. You can attach images that you copy or capture into the clipboard.
  • Link to other artifacts
    When entering a comment, you can link to another artifact by including that artifact's ID.
  • Track Subversion commits
    To be able to track Subversion commits related to an artifact, include the artifact ID when committing changes.
