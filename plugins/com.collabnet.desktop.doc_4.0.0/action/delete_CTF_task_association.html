Delete a task association

When a task artifact is no longer related to some other CollabNet TeamForge items, you can remove the association..

  1. Double-click a task in the CollabNet Sites view to open it in the artifact editor.
  2. Expand the Associations section, and select an association from the table.
  3. Click Delete. In that table row, a "-" icon appears adjacent to the association details.
    Note: The deletion is stored locally until you submit your changes to the server. If you have deleted the association in error, you can click Undelete to remove the deletion.
  4. Click Submit. The "-" icon that was displayed for that association, disappears.
    Note: If you do not submit your changes, the next time you open the application, the association is displayed with the "-" icon.
Tip: You can also delete an association in the CollabNet Sites Browser by expanding a task's Associations node and choosing the Delete association context menu option.
