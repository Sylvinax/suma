Work with Issue Tracker

In the CollabNet Sites view, you can define and run queries on Issue Tracker issues, create and edit issues, and add attachments to issues.

  • What is Issue Tracker?
    CollabNet's Issue Tracker is a variant of BugZilla, but a more generalized tool for tracking different kinds of activities.
  • Find Issue Tracker issues
    To search for Issue Trrcker issues, create and run a query.
  • Create an issue
    Add a new issue when you want to report a defect, or request and track a task, feature or enhancement.
  • Add an attachment
    You can attach text information from the clipboard, or files within or outside your Eclipse workspace. You can attach images that you copy or capture into the clipboard.
  • Link to other issues
    When entering a comment, you can link to another issue by including that issue number.
  • Access comments within an issue
    When you create an email message or enter an issue comment, you can add a link to a comment in the same or another issue.
