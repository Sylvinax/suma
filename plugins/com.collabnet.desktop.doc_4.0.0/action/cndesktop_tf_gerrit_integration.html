Work with Gerrit

When your site has the TeamForge Git integration installed, you will see a Gerrit node under your TeamForge project node in the CollabNet Sites view.

To be able to see the Gerrit node, your project must have at least one Git repository and you need permissions to view the source. If more than one Gerrit server is associated with your project, you will see a node for each server beneath the Gerrit node.
When you expand the Gerrit node, you will see pre-defined queries for Gerrit changes.
