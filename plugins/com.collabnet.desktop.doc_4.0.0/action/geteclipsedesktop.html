Get TeamForge connector functionality in your Desktop

To get TeamForge connector functionality in your Desktop, you need to install the CollabNet Connector Framework components in addition to the CollabNet Desktop component.

We recommend that you get Eclipse Classic 3.6.2.
  1. On the main menu bar, select Help > Install New Software .
  2. In the Install screen, provide this URL for the Work with field: http://downloads.open.collab.net/eclipse/update-site/e3.5
  3. In the list of components, select CollabNet Desktop and CollabNet Connector Framework and click Next.
  4. Follow the steps of the wizard and when the install is finished, restart Eclipse.
