Provision Lab Management hosts

Use Lab Management to access virtual or physical machines where you can build and test your code.

In the CollabNet Sites view, right-click a Lab Management host and choose an option under Host details.
  • To allocate a host to yourself or free up a system, select Summary.
  • To rebuild a host with a different profile, select Configuration.
  • For details about your host's resources are utilized, select Analytics.
  • For details about any action performed on a host, select Audit log. For example, when a host is rebuilt, details about the old and new profile, and how long it took to complete the process, are recorded in the Audit log.
  • For information about the build profile and version on your system, select Profile details.
The appropriate page opens up in the integrated browser where you can complete your task.
