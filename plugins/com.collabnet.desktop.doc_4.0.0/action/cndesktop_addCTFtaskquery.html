Add a task query

To work with your TeamForge tasks, start by gathering the relevant tasks with a query.

  1. In the CollabNet Sites view, select a level for your query:
    • Site-wide queries
    • Project-wide queries under a specific project
  2. Right-click and select New > Task Query .
  3. In the New Task Query window, enter a title for your query. Query title is a required field.
  4. Select the project whose tasks you want to query.
  5. Select other task attributes to include in the query.
  6. Click OK. Your query appears in the CollabNet Sites view.
