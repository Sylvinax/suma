Integrate with Review Board

Review Board is a popular code review tool available with TeamForge 6.2 (and later versions) as a fully-integrated add-on.

When Review Board is installed on your site, project administrators can make the application available for individual projects. If the Mylyn eReviewBoard plugin is installed, and the TeamForge project administrator has made Review Board available for your project, you will see the Review Board node in the CollabNet Sites view.
  1. In the CollabNet Sites view, expand Review Board.

    You will see the submitted, pending and discarded review requests assigned to you, assigned to everyone, and initiated by you.

  2. To respond to a review request, double-click it. The review request is opened in your browser. (Single sign-on is enabled.)
