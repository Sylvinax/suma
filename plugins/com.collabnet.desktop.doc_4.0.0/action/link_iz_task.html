Link to other issues

When entering a comment, you can link to another issue by including that issue number.

  1. Open an issue in the editor.
  2. In the New Comment field, enter a comment that includes the issue number of a related issue. For example, "4321".
  3. Click Submit. The issue whose ID you entered, is hyperlinked to the original one.
