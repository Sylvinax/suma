Find Issue Tracker issues

To search for Issue Trrcker issues, create and run a query.

  1. In the CollabNet Sites >view, click a CollabNet Issue Tracker project node.
  2. Right-click either of the following and select New > Issue Tracker query:
    • Project-wide queries
    • an issue type under Issue Tracker
    The New Tracker Query window appears.
  3. Enter a name for the query.
  4. Specify the parameters you want to include in the query and click OK.
  5. Expand the query in the CollabNet Sites view to see the corresponding issues.
