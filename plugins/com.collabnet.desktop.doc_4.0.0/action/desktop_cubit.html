Work with Lab Management

CollabNet Lab Management enables geographically distributed teams to provision and manage software development, build and test environments.

  • What is CollabNet Lab Management?
    CollabNet Lab Management is a Web-based, self-service interface that enables a geographically-distributed team to reserve a set of physical or virtual systems from a central pool of resources and rebuild these systems according to any pre-defined build-and-test configuration.
  • Get a Lab Management API key
    To access Lab Management Web Services, you must first get a Lab Management API key.
  • Add a Lab Management Manager site
    To access and manage your Lab Management hosts, add your Lab Management Manager site to the CollabNet Sites view.
  • Provision Lab Management hosts
    Use Lab Management to access virtual or physical machines where you can build and test your code.
  • Add an external SSH client
    You can specify and store a command to connect to a Linux or Solaris host using SSH.
  • Add an external RDP client
    You can specify and store a command to connect to a Windows host using the Remote Desktop Protocol.
  • Add a tunnel
    To add a tunnel, you must select a Lab Management host and specify the local and remote ports.
  • Manage tunnels
    In the Manage tunnels screen, you can add or remove tunnels, and open or close ones that you've created.
