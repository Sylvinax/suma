Issue Tracker attributes

This page describes the Issue Tracker fields available when you enter, view and modify issues.

To help you create effective issue reports, see the section titled Issue entry field descriptions for information on the following:
  • Version
  • Subcomponent
  • Platform
  • Operating System
  • Priority
  • Issue Type
  • Assign To
  • URL
  • Summary
  • Description
The Issue view page is similar to the Enter issue page and contains many of the same fields, but there are several important additions. See the section titled Viewing and modifying issues for help on these fields:
  • Target Milestone
  • Add/Remove CC
  • QA Contact
  • URL
  • Status whiteboard
  • Keywords
  • Attachments
  • Dependencies
  • Vote for this issue
  • Groups
  • Leave as NEW
  • Resolve issue
  • Reassign
  • View Issue Activity
  • Format For Printing
  • Format as XML
