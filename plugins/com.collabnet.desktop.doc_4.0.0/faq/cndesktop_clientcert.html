Does the CollabNet Desktop for Eclipse support client certificates for authorization to TeamForge?

Yes. The CollabNet Desktop (Development Build 2.3.3 and higher) supports login using a certificate file (PKCS12) or smartcard (PKCS11).

When adding a TeamForge site, you can specify whether to use basic authorization or a certificate. The certificate option lets you -
  • Select Certificate file (PKCS#12), and provide a client certificate file and passphrase.
  • Select Smartcard (PKCS#11) and select a certificate.
