What artifacts are synchronized?

In synchronizing a TeamForge project to a ScrumWorks Pro product, the following project mappings are available:

  • TeamForge Trackers <=> ScrumWorks Product Backlog Items (PBIs)
  • TeamForge Trackers <=> ScrumWorks Tasks
  • ScrumWorks Product => TeamForge Planning Folder (note that this is unidirectional)
  • ScrumWorks Release => TeamForge Planning Folder (note that this is unidirectional)
  • ScrumWorks Meta Data => Team Forge Meta Data (note that this is unidirectional)

    Meta Data includes User, Sprint, Team, and Theme data.
