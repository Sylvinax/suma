I've installed CollabNet Desktop but can't see any of the Mylyn features. What gives?

Use the file menu and see if you can select Window > Show View > Other > Mylyn. That should get you to the Mylyn views. If you do not see Mylyn listed in the Show View dialog, then most likely you are running on an older version of Java. You'll need to install JDK 1.5.x or higher for Mylyn to work.

For more information about using Mylyn in Eclipse Desktop see: What does Mylyn provide?
