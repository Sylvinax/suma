What is a Client Node?

Client Nodes refer to virtual or physical servers.

Lab Management distinguishes between the virtual and the physical in these instances:

  • Virtual machines can be migrated from one physical machine to another through the Lab Management Web User Interface. In contrast, physical machines cannot be migrated.
  • Physical machines can host virtual machines. In contrast, virtual machines cannot host other virtual machines.

The Client Nodes are set up for users to do their development work. Users can log in to and access the Client Nodes through the Lab Management Web User Interface or through shell accounts.
