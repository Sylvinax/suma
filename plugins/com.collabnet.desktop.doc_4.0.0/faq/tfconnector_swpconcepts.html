  • How are SWP Tasks synchronized with TeamForge artifacts?
    The connector supports bidirectional synchronization of ScrumWorks Pro Tasks and TeamForge artifacts. All fields synchronize bidirectionally except for the task's parent.
  • How are SWP Product Backlog Items synchronized with TeamForge artifacts?
    The connector supports bidirectional synchronization of ScrumWorks Pro Product Backlog Items and TeamForge artifacts. Some fields synchronize birectionally, while others only synchronize from ScrumWorks Pro to TeamForge.
  • How is SWP MetaData synchronized with TeamForge artifacts?
    All meta-data, which includes teams, sprints, themes, and users, is only synchronized from ScrumWorks Pro to TeamForge. Meta-data usage within Product Backlog Items and Tasks are not always a bidirectional synchronization.
  • Why do the user and resync user need to be dedicated connector users?
    The credentials for the connector user and resync user cannot be used to login to TeamForge or ScrumWorks. If changes are applied in either application by either the dedicated connector user or the resync user, the data will not be synchronized.
