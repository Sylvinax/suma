What is a Lab Management profile?

A Lab Management Profile is a machine and human-readable definition that can be used to construct a running operating system environment on a computer.

For example, a profile can include systems built by Jumpstart, Kickstart, or VMware images. Profile definitions are stored in XML format inside a Subversion repository.
