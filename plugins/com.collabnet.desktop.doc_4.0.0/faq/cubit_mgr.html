What is the Lab Management Manager?

The Lab Management Manager is the main node that Lab Management uses to run itself.

It performs many important functions.
  • Builds and provisions Client and Infrastructure Nodes
  • Collects usage data about each Client Node
  • Runs the Lab Management Web User Interface application that users employ to access the Client Nodes
  • Knows who is logged in at any time, anywhere
  • Allocates resources (such as profiles or hosts) for projects
  • Controls user access to Client Nodes
Note: There is one Lab Management Manager per Lab Management deployment. The Lab Management Manager must always be a physical machine.
