What is Project Tracker?

Project Tracker is a highly configurable artifact tracking tool that meets the needs of different types of projects as well as individual users.

You can configure each implementation for the workflow processes of a project through the definition of artifact types and attributes. The user interface is designed to be accessible and constant: once you become familiar with the look and feel, you will find new processes quick to master.

Project Tracker provides these tools for artifact management:

  • Artifact entry: The artifact entry process is designed to help you create meaningful artifacts and avoid duplication. Administrators can configure artifact entry to check for duplicates before an artifact can be submitted. If any duplicates are found, you can review potential duplicates, add comments to similar artifacts or modify your artifact before committing it to the database.
  • Templates for artifact entry: You can save frequently entered artifact criteria in templates for speedy entry.
  • Artifact query: You can access artifacts in the database using the query feature. You can find, access and modify artifacts individually. The advanced query feature allows you to create detailed artifact searches, and limit your search to specific date or artifact number ranges. Frequently used queries can be saved and run.
  • Reports: The robust reports interface allows you to create and save reports. Reports provide customizable metrics.
  • Email notifications: As important events occur, email messages are generated and sent to all parties associated with the event. Alert email messages contain links to views of the event.
